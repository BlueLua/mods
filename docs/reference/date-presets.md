# Date Presets

Preset format aliases for English locales, used by the [`date`](../api/date.md)
module.

| Alias  | Expands to                  |
| ------ | --------------------------- |
| `LT`   | `h:mm A`                    |
| `LTS`  | `h:mm:ss A`                 |
| `L`    | `MM/DD/YYYY`                |
| `LL`   | `MMMM D, YYYY`              |
| `LLL`  | `MMMM D, YYYY h:mm A`       |
| `LLLL` | `dddd, MMMM D, YYYY h:mm A` |
| `l`    | `M/D/YYYY`                  |
| `ll`   | `MMM D, YYYY`               |
| `lll`  | `MMM D, YYYY h:mm A`        |
| `llll` | `ddd, MMM D, YYYY h:mm A`   |
