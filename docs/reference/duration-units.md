Supported date and time units, used by the [`date`] and [`duration`] modules.

| Unit          | Supported Aliases                       |
| :------------ | :-------------------------------------- |
| `year`        | `y`, `years`, `year`                    |
| `quarter`     | `q`, `quarters`, `quarter`              |
| `month`       | `M`, `months`, `month`                  |
| `week`        | `w`, `weeks`, `week`                    |
| `day`         | `d`, `days`, `day`                      |
| `hour`        | `h`, `hours`, `hour`                    |
| `minute`      | `m`, `mins`, `min`, `minutes`, `minute` |
| `second`      | `s`, `secs`, `sec`, `seconds`, `second` |
| `millisecond` | `ms`, `milliseconds`, `millisecond`     |
